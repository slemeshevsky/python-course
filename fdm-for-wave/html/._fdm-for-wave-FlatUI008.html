Задача 12: Моделирование волн на неоднородной струне

1) Написать функцию-решатель смешанной задачи $$ \begin{align} \tag{60} \rho(x) \pdd{2}{u}{t} = S \pdd{2}{u}{x}, \quad 0 < t \leq T, \quad 0 < x < l, \quad S = \textrm{const},\\ \tag{61} u(0, t) = u(l,t) = 0,\quad 0 < t \leq T,\\ \tag{62} u(x, 0) = I(x), \quad \pd{u(x,0)}{t} = V(x), \quad 0 < x < l, \end{align} $$ где $$ \begin{equation} \tag{63} \rho(x) = \begin{cases} \rho_1, & 0 < x < l/2;\\ \rho_2, & l/2 < x < l. \end{cases} \end{equation} $$ 2) Написать тестовые функции для проверки реализации алгоритма.

3) Реализовать сохранение результатов на диск.

4) Реализовать анимацию для визуализации результатов.
