Упражнения¶

Exercise 6: Нахождение аналитического решения волнового уравнения с затуханием¶

Рассмотрим волновое уравнение с затуханием (44). Цель упражнения — найти точное решение задачи с затуханием. Отправная точка — стоячая волна из Exercise 1: Моделирование стоячей волны. Необходимо включить множитель затухания \(e^{-ct}\), а также синусоидальной и косинусоидальные компоненты по времени:

\[u_e(x,t) = e^{-\beta t} \sin kx \left( A\cos\omega t + B\sin\omega t\right).\]

Найти \(k\), используя граничные условия \(u(0,t) = u(l,t) = 0\). Затем используйте уравнение для нахождения ограничений на \(\beta\), \(\omega\), \(A\) и \(B\). Сформулируйте полную начально-краевую задачу и ее решение. Filename: damped_waves.

Problem 7: Анализ симметричных граничных условий¶

Рассмотрим простую волну-“площадку”, где \(\Omega = [-l, l]\) и

\[\begin{split}I(x) = \begin{cases} 1, & x \in [-\delta, \delta],\\ 0, \textrm{иначе}, \end{cases}\end{split}\]

для некоторого числа \(0 < \delta < l\). Второе начальное условие — \(\frac{\partial u(x,0)}{\partial t} = 0\). Предположим также, что отсутствует правая часть \(f\). В качестве граничных условий выберем однородные условия Дирихле. Решение данной задачи симметрично относительно \(x = 0\). Это означает, что мы можем проводить моделирование только на половине области \([0,l]\).

1) Показать почему симметричным граничным условием будет \(\frac{\partial u}{\partial x} = 0\) при \(x = 0\).

2) Выполнить расчет задачи для волнового уравнения на отрезке \([-l,l]\). После этого, используйте симметричность решения и выполните расчет в половине области \([0,l]\), используя симметричное граничное условие при \(x = 0\). Сравните два решения и удостоверьтесь в том, что они совпадают.

3) Докажите симметричность решения, сформулировав начально-краевую задачу и показав, что если \(u(x,t)\) — решение задачи, то и \(u(-x,t)\) также является решением этой задачи.

Hint. Симметричность функции относительно \(x_0 = 0\) означает, что \(f(x_0+h) = f(x_0 - h)\).

Filename: waved1d_symmetric.

Exercise 8: Имульс через слоистую среду¶

Используйте функцию pulse из wave1d_dn_vc.py для того, чтобы исследовать распространение импульса, чей начальный пик расположен в \(x=0\), через две среды с разными скоростями распространения волн. Скорость распространения волны в левой области положить равной \(1\), а в правой области — \(s_f\). Показать, что происходит для разных импульсов при \(N_x = 40, 80, 160\) и \(s_f = 2, 4\). Моделирование проводить до \(T = 2\). Filename: pulse1d.py.

Exercise 9: Объяснение почему возникают численные шумы¶

Эксперименты из Exercise 8: Имульс через слоистую среду показывают существенные численные шумы (осцилляции) в виде нифизичных колебаний, особенно для \(s_f = 4\) и импульса площадки (plug) или half-cosinehat импульса. Шумы менее видны для Гауссового импульса. Выполните варианты расчетов для импульсов plug и half-cosinehat при \(s_f = 1\), \(\gamma = 0.9, 0.25\) и \(N_x = 40, 80, 160\). Используйте отношение численной дисперсии для обоснования результатов. Filename: pulse1d_analysis.

Problem 11: Реализация условий открытых границ¶

Для того, чтобы дать возможность волне покинуть вычислительную область и перемещаться без искажений через границу \(x=l\), в одномерной модели можно наложить следующее условие, называемое условием излучения или условием открытой границы:

\[\tag{52} \frac{\partial u}{\partial t} + c \frac{\partial u}{\partial x} = 0.\]

Параметр \(c\) — скорость распространения волны.

Показать, что (52) допускает решение \(u = g_R(x - ct)\) (волна, бегущая вправо), но не \(u = g_L(x + ct)\) (волна, бегущая влево).

Соответствующее условие открытой границы при \(x=0\) имеет вид

\[\tag{53} \frac{\partial u}{\partial t} - c \frac{\partial u}{\partial x} = 0.\]

1) Естественная идея аппроксимации граничного условия (52) в конечном узле с индексом \(i = N\) заключается в использовании центральных разностных производных по пространственной и временной перемменной:

\[\tag{54} y_{\mathring{t},N}^n + c y_{\mathring{x},N}^n = 0.\]

Избавиться от фиктивного значения \(y_{N+1}^n\), используя аппроксимацию уравнения в этой же точке.

Уравнение для \(y_i^1\) также нарушается, мы можем использовать условие \(u_N^1=0\), так как волна еще не достигла правой границы.

2) Более удобная реализация условия открытой границы при \(x = l\) может основываться на явной дискретизации

\[\tag{55} y_{t,N}^n + c y_{\bar{x},N}^n = 0.\]

Из этого уравнения можно найти \(y_N^{n+1}\) и его как условие Дирихле. Однако, в этом случае аппроксимация будет иметь первый порядок.

Реализовать эту схему для волнового уравнения \(\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}\) на отрезке \([0,l]\) с граничным условием \(\frac{\partial u}{\partial x} = 0\) при \(x = 0\), условием (52) при \(x = l\), и начальным возмущением в центре отрезка, например, профиль-площадка вида

\[\begin{split}u(x,0) = \begin{cases} 1, & l/2-\ell \leq x \leq l/2+\ell,\\ 0, &\textrm{в противном случае}. \end{cases}\end{split}\]

Обратите внимание на то, что начальный профиль распадается на две площадки, волна, бегущая влево, отражается от границы \(x = 0\), и обе волны уходят за границу \(x = l\), оставляя в итоге решение \(u = 0\) на \([0, l]\). Используйте число Куранта равное единице, что позволит получить точное решение. Создать анимированный график.

3) Добавить возможность задания либо условия \(\frac{\partial u}{\partial x} = 0\), либо условия открытой границы при \(x=0\). Последнее условие аппроксимировать как

\[y_{t,0}^n - c y_{x,i}^n = 0,\]

дающее возможность явного вычисления \(y_0^{n+1}\).

Реализацию можно протестировать на функции Гаусса как начальном условии:

\[g(x; m, s) = \frac{1}{\sqrt{2\pi} s}e^{-\frac{(x-m)^2}{2s^2}}.\]

Выполнить два теста: 1. Возмущение в центре области, \(I(x) = g(x; l/2, s)\) и открытой левой

границей. 1. Возмущение на левом конце области, \(I(x) = g(x; 0, s)\) и условием

симметрии \(\frac{\partial u}{\partial x} = 0\) на левой границе.

Создать юнит-тесты для обоих случаев, проверяющие, что решение равно нулю после того, как волна покинет область.

Hint. Модифицируйте солвер из wave1d_n.py.

Filename: wave1d_open_bc.py.
