Разностные схемы для многомерного волнового уравнения¶

Рассмотрим использование разностных схем для двумерного (2D) и трехмерного (3D) уравнения.

Многомерное волновое уравнение¶

Общее волновое уравнение в пространстве размерности \(d\) с постоянной скоростью распространения волны \(c\) можно записать в компактном виде

\[\tag{56} \frac{\partial^2 u}{\partial t^2} = c^2 \Delta u \quad \textrm{для}\ \boldsymbol{x} \in \Omega\subset\mathbb{R}^d, \quad t\in (0,T],\]

где \(\Delta\) оператор Лапласса:

\[\Delta u = \sum_{\alpha=1}^{d} \frac{\partial^2 u}{\partial x_\alpha^2}\]

Во многих приложениях коэффициенты уравнения зависят от точки пространства. В этом случае волновое уравнение имеет вид:

\[\tag{57} \varrho(\boldsymbol{x}) \frac{\partial^2 u}{\partial t^2} = \sum_{\alpha=1}^{d}\pd{}{x_\alpha}\left( k_\alpha(\boldsymbol{x}) \frac{\partial u}{\partial x_\alpha} \right) + f(\boldsymbol{x})\quad \textrm{для}\ \boldsymbol{x}\in\Omega\subset\mathbb{R}^d,\quad t \in (0,T].\]

В каждой точке границы \(\partial\Omega\) (области \(\Omega\)) необходимо задать одно условие, включающее в себя неизвестное решение \(u\). Граничные условия задаются трех видов:

  1. \(u\) задано (\(u = 0\) или заданная функции от \(t\) в точках границы,
например, моделирующее входящую волну);
  1. \(\displaystyle\frac{\partial u}{\partial \mathbf{n}} = \mathbf{n}\cdot {\mathrm{grad}\,u}\) задана (ноль для
отражающих границ)
  1. открытое граничное условие (также называемое условием
излучения). См. Problem 11: Реализация условий открытых границ.

Все приведенные уравнения со второй производной по времени должны быть дополнены двумя начальными условиями:

  1. \(u = I\);
  2. \(\displaystyle\frac{\partial u}{\partial t} = V\).

Сетка¶

Введем сетки по времени и по пространству. Сетка с постоянным шагом по времени

\[\bar{\omega}_{\tau} = \{ t_n = n \tau,\ n = 0,1, \ldots, N_t, \ N_t\tau = T \}.\]

Разностные схемы легко реализовать в простых прямоугольных областях. Более сложные формы областей требуют существенно более совершенных методов и усилий по реализации. В прямоугольных областях можно ввести отдельные сетки по каждому направлению:

\[\bar{\omega}_{\alpha} = \{ x_{\alpha}^{(i_\alpha)} = i_\alpha h_\alpha, \ i_\alpha = 0, 1, \ldots, N_\alpha, \ N_\alpha h_\alpha = l_\alpha \}, \quad k = 1, 2, \ldots, d.\]

Также можно ввести сетки (с использованием множеств индексов)

\[\begin{split}\begin{split} \omega_\alpha &= \{ x_{\alpha}^{(i_\alpha)} = i_\alpha h_\alpha, \ i_\alpha \in \mathcal{I}_{x_\alpha}^i \},\\ \omega_\alpha^{+} &= \{ x_{\alpha}^{(i_\alpha)} = i_\alpha h_\alpha, \ i_\alpha \in \mathcal{I}_{x_\alpha}^+ \},\\ \omega_\alpha^{-} &= \{ x_{\alpha}^{(i_\alpha)} = i_\alpha h_\alpha, \ i_\alpha \in \mathcal{I}_{x_\alpha}^- \}. \end{split}\end{split}\]

Таким образом сетка по пространству в области \(\Omega\):

\[\begin{split}\begin{split} \bar\omega_h &= \prod_{\alpha=1}^{d} \bar\omega_\alpha = \{ \boldsymbol{x} = \boldsymbol{x}^{(i_1i_2\dots i_d)} = (x_1^{i_1}, x_2^{(i_2)}, \dots, x_d^{(i_d)}), \ x_\alpha^{(i_\alpha)} \in \omega_\alpha, \ k = 1, 2, \dots, d\},\\ \omega_h &= \prod_{\alpha=1}^{d} \omega_\alpha. \end{split}\end{split}\]

Для значений приближенного решения в узлах сетки будем использовать следующие обозначения:

\[\begin{split}\begin{split} y^n &= y_{(i_1i_2\dots i_d)}^n = y(\boldsymbol{x}^{(i_1i_2\dots i_d)},t_n), \\ y_{(\pm 1_\alpha)}^n &= y_{(i_1\dots i \pm 1\dots i_d)}^n = y(\boldsymbol{x}^{(i_1\dots i_\alpha \pm 1\dots i_d)},t_n) = y(x_1^{(i_1)},\dots, x_\alpha^{(i_\alpha)}+h_\alpha, \dots, x_d^{(i_d)},t_n), \quad n\in \mathcal{I}_t,\ i_\alpha \in \mathcal{I}_{x_\alpha}, \ k = 1, 2, \ldots, d. \end{split}\end{split}\]

Явная разностная схема¶

Многомерные уравнения аппроксимируются путем сборки одномерных дискретизаций, так как многомерное уравнение просто содержит слагаемые того же типа, что и одномерное уравнение.

Дискретизация уравнения¶

Уравнение (57) аппроксимируем явной разностной схемой (используем безындексные обозначения):

\[\tag{58} \varrho y_{\bar{t}t} = \sum_{\alpha=1}^{d} (a_\alpha y_{\bar{x}_\alpha})_{x_\alpha} + \varphi, \quad \boldsymbol{x} \in \omega,\ t \in \omega_\tau.\]

Шаблонные функционалы для коэффициентов и правой части используем, например, следующие:

\[\begin{split}\begin{split} a_\alpha &= a_\alpha(\boldsymbol{x}^{(i_1\dots i_\alpha+0.5\dots i_d)}) = a(x_1, \dots, x_{\alpha}+0.5h_\alpha, \dots, x_d), \quad \alpha = 1, 2, \ldots, d, \\ \varphi &= f(\boldsymbol{x}, t),\quad \boldsymbol{x} \in \omega, \ t \in \omega_\tau. \end{split}\end{split}\]

Выражая \(y^{n+1}\) (см. обозначения), получим рекуррентную формулу:

\[\tag{59} y^{n+1} = -y^{n-1} + 2 y^{n} + \frac{1}{\varrho} \sum_{\alpha=1}^{d} \frac{1}{h_k^2} \left(a_{\alpha(+1_k)} \left(y_{(+1_k)}^n - y^n\right) - a_{\alpha} \left(y^n - y_{(-1_k)}^n \right)\right) + \tau^2 \varphi.\]

Также необходимо получить специальную формулу для \(y^{1}\), объединив (59) при \(n=0\) и дискретное начальное условие, которое дает \(y^{-1} = y^{1} - 2\tau V\).

Обработка граничных условий Дирихле¶

Разностные схемы (58) и (59) справедливы для внутренних узлов пространственной сетки. После вычисления значений во внутренних узлах, нужно установить заданные значения \(y\) во всех граничных узлах сетки.

Дискретизация условий Неймана¶

Условие \(\displaystyle\frac{\partial u}{\partial \mathbf{n}} = 0\) в одномерном случае мы аппроксимировали центральной разностной производной, а затем исключали значение \(y\) в фиктивном узле вне сетки, используя аппроксимацию уравнения вблизи границы. Другой способ заключался в использовании мнимых ячеек. Точно такие же подходы можно использовать в многомерном случае.

Рассмотрим условие \(\displaystyle\frac{\partial u}{\partial \mathbf{n}} = 0\) на границе \(x_\alpha = 0\) прямоугольной области \(\Omega = \prod_{k=1}^{d}[0, l_k]\). Направление нормали к этой границе — это направление в отрицательную часть \(x_\alpha\). Следовательно,

\[\frac{\partial u}{\partial \mathbf{n}} = - \frac{\partial u}{\partial x_\alpha}.\]

Положим

\[-y_{\mathring{x}_\alpha} = 0 \quad \Rightarrow \quad \frac{y_{i_1\dots 1\dots i_d}-y_{i_1\dots -1\dots i_d}}{2h_\alpha} = 0.\]

Отсюда следует, что \(y_{i_1\dots 1\dots i_d} = y_{i_1\ldots -1\ldots i_d}\). Затем, подставляя \(y_{i_1\dots 1\dots i_d}\) вместо \(y_{i_1\ldots -1\ldots i_d}\) в рекуррентную формулу (59), записанную в граничных узлах \(i_\alpha = 0\), получаем формулу для \(y_{i_1\dots 0 \dots i_d}^{n+1}\) аналогично как в одномерном случае.

Из приведенных рассуждений мы можем использовать такой подход: общее рекуррентное соотношение (59) может использоваться и в граничных узлах \(i_\alpha = 0\), если заменить \(i_\alpha - 1\) на \(i_\alpha + 1\). Такой поход полезен при программной реализации. Детали для одномерного случая можно увидеть в разделе Программная реализация условий Неймана.
