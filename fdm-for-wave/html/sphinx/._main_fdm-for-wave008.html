Problem 12: Моделирование волн на неоднородной струне¶

1) Написать функцию-решатель смешанной задачи

\[\tag{60} \rho(x) \frac{\partial^2 u}{\partial t^2} = S \frac{\partial^2 u}{\partial x^2}, \quad 0 < t \leq T, \quad 0 < x < l, \quad S = \textrm{const},\]
\[\tag{61} u(0, t) = u(l,t) = 0,\quad 0 < t \leq T,\]
\[\tag{62} u(x, 0) = I(x), \quad \frac{\partial u(x,0)}{\partial t} = V(x), \quad 0 < x < l,\]

где

\[\begin{split}\tag{63} \rho(x) = \begin{cases} \rho_1, & 0 < x < l/2;\\ \rho_2, & l/2 < x < l. \end{cases}\end{split}\]

2) Написать тестовые функции для проверки реализации алгоритма.

3) Реализовать сохранение результатов на диск.

4) Реализовать анимацию для визуализации результатов.

\[ \begin{align}\begin{aligned}\begin{gather}\\\begin{split} \rho \frac{\partial^2 u}{\partial t^2} + b \frac{\partial u}{\partial t} = S \frac{\partial^2 u}{\partial x^2}, \quad 0 < t \leq T, \quad 0 < x < l, \quad \rho, S = \textrm{const}, \quad b \geq 0,\\\end{split}\\\begin{split} u(0, t) = u(l,t) = 0,\quad 0 < t \leq T,\\\end{split}\\ u(x, 0) = I(x), \quad \frac{\partial u(x,0)}{\partial t} = V(x), \quad 0 < x < l, \end{gather}\end{aligned}\end{align} \]

2) Написать тестовые функции для проверки реализации алгоритма.

3) Реализовать сохранение результатов на диск.

4) Реализовать анимацию для визуализации результатов.

Problem 14: Моделирование упругих волн в стержне¶

1) Написать функцию-решатель смешанной задачи

\[ \begin{align}\begin{aligned}\begin{gather}\\\begin{split} \rho \frac{\partial^2 u}{\partial t^2} = \frac{\partial }{x}\left( E \frac{\partial u}{\partial \partial x} \right), \quad 0 < t \leq T, \quad 0 < x < l, \quad \rho = \textrm{const},\\\end{split}\\\begin{split} \frac{\partial u(0, t)}{\partial x} = 0,\quad \frac{\partial u(l,t)}{\partial x} = \sigma_{xx}(t), \quad 0 < t \leq T,\\\end{split}\\ u(x, 0) = 0, \quad \frac{\partial u(x,0)}{\partial t} = V(x), \quad 0 < x < l, \end{gather}\end{aligned}\end{align} \]

где

\[\begin{split}\tag{64} \sigma_{xx}(t) = \begin{cases} S/E, & 0 < t < t_s,\\ 0, & t > t_s \end{cases}\end{split}\]

2) Написать тестовые функции для проверки реализации алгоритма.

3) Реализовать сохранение результатов на диск.

4) Реализовать анимацию для визуализации результатов.

Problem 16: Моделирование двумерной сейсмической волны¶

1) Написать функцию-решатель смешанной задачи

\[ \begin{align}\begin{aligned}\begin{gather}\\\begin{split} \frac{\partial^2 p}{\partial t^2} = c_z^2 \frac{\partial^2 p}{\partial z^2} + c_h^2(\frac{\partial^2 p}{\partial x^2} + \frac{\partial^2 p}{\partial y^2}) + s(x,y,t), \quad 0 < t \leq T, \quad (x,y) \in [0,l_1]\times [0, l_2],\\\end{split}\\\begin{split} p(x,y) = p_0, \quad x = 0, \ x= l_1, \ y = 0, \ y = l_2,\\\end{split}\\ p(x, 0) = p_0, \quad \frac{\partial u(x,0)}{\partial t} = V(x), \quad 0 < x < l, \end{gather}\end{aligned}\end{align} \]

где

\[\tag{65} s(x,y,t) = \sin(t) e^{\frac{(x-x_c)^2 + (y-y_c)^2}{2\sigma^2}},\]

2) Написать тестовые функции для проверки реализации алгоритма.

3) Реализовать сохранение результатов на диск.
